Iambic Meter in Russian verse: examples

Here is an example of a poem written in iambic tetrameter.
A fragment from Chapter 3 of Eugene Onegin (Alexander Pushkin)
I can foresee a complication
x X | x X | x x | x X x
My country’s honor to defend
x X | x X | x x | x X
I’ll have to furnish a translation
x X | x X | x x | x X
Of Tanya’s letter in the end.
x X | x X | x x | x X

The lead female character in this novel, Tanya, writes a love letter to Eugene. She is a noble woman and her first language is French. Hence, the narrator has to translate her letter for the readers.
